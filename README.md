# simple_handmade_shop It`s project - internet shop. This application is uses a database, authentication, Js, CSS. In the future, I will deploy online.
What kind of project is this? 
This project uses MVC and Builder patterns, Entity Framework. The project also has authentication, work with XLWorkbook, MailMessage, SmtpClient libraries and classes. The administrator receives a notification when a product is purchased. This project is integrated payment system.
If you want to view your statistics, the project has a dashboard.
